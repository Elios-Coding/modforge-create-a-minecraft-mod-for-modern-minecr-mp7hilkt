# Global TNT Power Command

Adds an operator-only /tntpower <multiplier> command that globally scales vanilla TNT explosion strength via a server-side multiplier.

Generated by **ModForge AI**.

- Minecraft: `1.21.11`
- Loader: Fabric `0.18.4`
- Fabric API: `0.141.4+1.21.11`

## Build

```bash
./gradlew build
```

The compiled `.jar` will be placed in `build/libs/`. GitHub Actions builds it automatically on push and publishes it to Releases.
